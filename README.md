# BlueTech Education - Online Course Platform

A modern, high-end online course platform UI built with React, Tailwind CSS, and Lucide icons.

## Features
- **Main Course Catalog:** A grid of courses with details.
- **Student Learning Area:** A full classroom experience with:
  - Lessons sidebar.
  - Video player placeholder.
  - **AI Mentor:** A futuristic chat panel for student assistance.
  - Task completion system.
- **RTL Support:** Fully optimized for Hebrew.
- **Responsive Design:** Works on all devices.

## Tech Stack
- React
- Tailwind CSS
- Lucide Icons
- Vite

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
